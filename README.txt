Roomer
=================

Simple calendar for conference rooms booking. Gives ability to make a reservation for two weeks.

== Features

* rooms reservations system for following two weeks, each room has own box in graphic representation of each day
* rooms listing with names and numbers identification
* authentication system with simple registeration process (using restful_authentication plugin)
